## 배열이란
"배열은 같은 타입의 여러 변수를 하나의 묶음으로 다루는 것"
중요한 점 - 같은 타입이어야 한다. 

배열을 사용하지 않는다면 아래와 같이 5개의 변수를 선언해야 한다.


int score1, score2, score3, score4, score 5;

하지만 배열을 사용한다면 아래와 같이 간단히 처리 할 수 있다


int [] score = new int [5];
